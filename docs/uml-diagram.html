OOP Principles Demonstrated

Inheritance

Customer and Admin extend abstract User. Runtime polymorphism via User* pointer calling derived methods.

Abstraction

User is abstract with pure virtual functions (displayMenu, getRole). Cannot be instantiated directly.

Encapsulation

All data members private. Controlled access through getters/setters. Cart and Order hide internal item vectors.

Composition

Cart and Order own CartItem instances. Items are created/destroyed with their container (filled diamond).

Aggregation

Inventory manages Product references. Products persist independently via file I/O (empty diamond).

Exception

ShopException extends std::exception. Used for out-of-stock, invalid product ID, and authentication errors.
